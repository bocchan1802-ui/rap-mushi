# 🎤 ラップムしクローン

虫たちがラップバトルをするリズムゲームのクローンアプリです。

## 🎵 デモ

- [オンラインデモ](https://bocchan1802-ui.github.io/rap-mushi/) (準備中)

## 📸 スクリーンショット

## 🎮 遊び方

1. 🎵 **パッドをタップ** - ラップフレーズが再生されます
2. 🔴 **録音ボタン** - 自分のラップを録音開始/停止
3. ▶️ **再生ボタン** - 録音したラップを再生
4. 🗑️ **消去ボタン** - 録音をクリア
5. ❓ **ヘルプボタン** - 遊び方を表示
